## Project Structure

This is a JetBrains IDE plugin project for CodeIgniter Load Analyzer.

## Build Commands

- Build plugin: `./gradlew buildPlugin`
- Run test IDE: `./gradlew runIde`
- Verify plugin: `./gradlew verifyPlugin`
- Clean build: `./gradlew clean`
